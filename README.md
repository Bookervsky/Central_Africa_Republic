# Central African Republic Spatiotemporal Data Analysis
This repository aggregates spatiotemporal data for the Central African Republic at the prefecture (province) level. Those data will be used for conducting spatiotemporal analysis in the future.  <br>

## Aggregated Data
The aggregated data files are provided in Geojson and CSV format, located in [data/Aggregated_prefecture](data/Aggregated_prefecture)

## Data Sources
For description of the data, please refer to: [Format description PDF](https://download.geofabrik.de/osm-data-in-gis-formats-free.pdf)   <br>
To download the source data, please visit: [Central African Republic](https://download.geofabrik.de/africa/central-african-republic.html#)
